# Calculadora Simples 
Uma Calculadora Simples, onde ela executa operações com 2 numeros. Possui as seguintes operações disponiveis: +(adição),-(subtração),\*(multiplicação), /(divisão real e inteira) e \*\*(potencialização). 

**Qualquer adição ou erro que possa ter no código fico aberto a sugestões e correções na aba de issues.**
